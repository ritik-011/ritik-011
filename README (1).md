## 🔬 Research Publication

<div align="center">

> 📄 **"Hybrid Framework for Student Performance Prediction in Virtual Learning Environments"**

</div>

| 🏷️ | Details |
|-----|---------|
| 👨‍💻 **Authors** | Ritik, Anika — Gautam Buddha University, Greater Noida |
| 🎯 **Objective** | Predict at-risk students in Virtual Learning Environments using a Hybrid CNN + Random Forest model |
| 📦 **Dataset** | OULAD (Open University Learning Analytics Dataset) — 32,593 students, 7 course modules |
| 🧠 **Model** | 1D-CNN for feature extraction + Random Forest classifier with weighted probability fusion |
| ⚖️ **Imbalance Handling** | SMOTE + Inverse-frequency class weighting |
| 🏆 **Best Accuracy** | **97.55%** on CCC module — outperformed all baseline ML & DL models |
| 📊 **Avg Accuracy** | **92.80%** across all 7 OULAD course modules |

**🔧 Tech Stack Used:**

![Python](https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=python&logoColor=white)
![TensorFlow](https://img.shields.io/badge/TensorFlow-FF6F00?style=flat-square&logo=tensorflow&logoColor=white)
![Scikit-learn](https://img.shields.io/badge/Scikit--Learn-F7931E?style=flat-square&logo=scikit-learn&logoColor=white)
![Pandas](https://img.shields.io/badge/Pandas-150458?style=flat-square&logo=pandas&logoColor=white)
![NumPy](https://img.shields.io/badge/NumPy-013243?style=flat-square&logo=numpy&logoColor=white)

**💡 Key Highlights:**
- 🧬 Novel hybrid architecture combining **deep learning + ensemble learning**
- 📈 Outperformed ResNet, TabTransformer, DNN, SVM, LR, NB baselines
- 🎓 Real-world applicability for **early warning systems** in e-learning platforms
- 🔁 Validated across **7 heterogeneous course modules** for generalizability
